# 專案介紹
基於對岸火紅的微信小程序來開發的仿京東購物App，以微信開發者工具進行開發，圖片素材取自網路。
# 使用技術
WXML+WXS
# 運行方式
使用微信開發者工具導入資料夾並運行編譯
